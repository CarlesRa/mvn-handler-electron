# MVN-HANDLER
Mvn-Handler is a tool to generate and copy your war files into tomcat.

[![npm version]()
[![License: ISC]()

## Install

Download repo:
https://github.com/CarlesRa/mvn-handler-electron

Run:
```shell
npm i
npm run make
```
Go to out folder in app root directory, and execute .exe app file to install app.
